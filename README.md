# Bootstrap 4 Portfolio Website

A simple portfolio website built using Bootstrap 4 and custom CSS.

## Features

- Responsive navigation
- Hero section
- About section
- Contact section
- Custom styling
- Mobile-friendly design

## Technologies Used

- HTML5
- CSS3
- Bootstrap 4.5.2
- jQuery 3.5.1

## Setup

1. Clone the repository
2. Open `index.html` in your web browser
